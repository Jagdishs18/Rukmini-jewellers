Accessibility Statement

Last Updated: March 2025

RUKMINIHALL JEWELLERS is committed to ensuring digital accessibility for people with disabilities. We are continually improving the user experience for everyone and applying the relevant accessibility standards.

Measures to Support Accessibility

RUKMINIHALL JEWELLERS takes the following measures to ensure accessibility:

  • Include accessibility as part of our mission statement
  • Integrate accessibility into our design process
  • Use clear, simple language throughout the website
  • Provide text alternatives for non-text content
  • Ensure sufficient color contrast

Conformance Status

The Web Content Accessibility Guidelines (WCAG) define requirements for designers and developers to improve accessibility for people with disabilities. We aim to conform to WCAG 2.1 Level AA.

Feedback

We welcome your feedback on the accessibility of RUKMINIHALL JEWELLERS website. Please let us know if you encounter accessibility barriers:

  • Phone: +91 8156 123456
  • Email: accessibility@rukminihalljewellers.com
  • Visitor address: Main Road, Channarayapatna

We try to respond to feedback within 3 business days.

Technical Specifications

Accessibility of the RUKMINIHALL JEWELLERS website relies on the following technologies to work with the particular combination of web browser and any assistive technologies or plugins installed on your computer:

  • HTML
  • CSS
  • JavaScript

Assessment Approach

RUKMINIHALL JEWELLERS assessed the accessibility of this website using internal tools and periodic external audits.
